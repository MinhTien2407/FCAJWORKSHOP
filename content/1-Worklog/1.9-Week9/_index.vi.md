### Kết quả đạt được tuần 9:

* Hiểu được mô hình quản lý danh tính và phân quyền trong AWS.

* Thiết lập hệ thống Single Sign-On:
  * Cấu hình AWS IAM Identity Center
  * Quản lý tài khoản người dùng tập trung
  * Đơn giản hóa quá trình đăng nhập
  * ...

* Thực hành quản lý quyền truy cập:
  * Thiết lập IAM Permission Boundary
  * Giới hạn quyền tối đa cho IAM User và IAM Role
  * Áp dụng nguyên tắc Least Privilege
  * ...

* Cấu hình giới hạn Assume Role:
  * Thiết lập Condition cho IAM Role
  * Kiểm soát quyền chuyển đổi Role
  * Tăng cường bảo mật truy cập
  * ...

* Đánh giá bảo mật hệ thống:
  * Kiểm tra Security Standards
  * Phát hiện các vấn đề bảo mật
  * Đưa ra khuyến nghị khắc phục
  * ...

* Bảo vệ ứng dụng với AWS WAF:
  * Tạo Web ACL
  * Thiết lập Security Rules
  * Ngăn chặn các cuộc tấn công Web phổ biến
  * ...

* Hiểu được các phương pháp bảo vệ tài nguyên AWS theo các Best Practices.

* Nâng cao kỹ năng quản trị bảo mật và kiểm soát truy cập trong môi trường Cloud.

* Hoàn thành workshop về AWS Security:
  * Thiết lập Single Sign-On
  * Quản lý IAM Permission Boundary
  * Kiểm tra bảo mật với AWS Security Hub
  * Bảo vệ ứng dụng bằng AWS WAF
  * ...
