## Tuần 11 : Data Lake, Data Analytics & Business Intelligence on AWS 
### A. Mục tiêu

* Xây dựng kiến trúc Data Lake trên nền tảng AWS để lưu trữ và quản lý dữ liệu tập trung.
* Thực hành xây dựng quy trình xử lý dữ liệu từ thu thập, lưu trữ đến phân tích và trực quan hóa.
* Làm quen với các dịch vụ phân tích dữ liệu trên AWS như AWS Glue, Amazon Athena và Amazon QuickSight.
* Xây dựng Dashboard trực quan phục vụ cho việc phân tích và hỗ trợ ra quyết định.

### C. Kết quả đạt được

* Hiểu được kiến trúc tổng thể của một **Data Lake** trên nền tảng AWS.

* Thực hành xây dựng Data Lake với dữ liệu thực tế:
  * Lưu trữ dữ liệu trên Amazon S3
  * Xây dựng Data Pipeline
  * Thực hiện ETL với AWS Glue
  * ...

* Nắm được quy trình quản lý dữ liệu trên AWS:
  * Data Ingestion
  * Data Catalog
  * Data Transformation
  * Data Query
  * ...

* Sử dụng Amazon Athena để:
  * Truy vấn dữ liệu bằng SQL
  * Phân tích dữ liệu trực tiếp trên Amazon S3
  * Tối ưu chi phí phân tích dữ liệu
  * ...

* Làm quen với các dịch vụ Data Analytics trên AWS:
  * AWS Glue
  * Amazon Athena
  * Amazon S3
  * Amazon QuickSight
  * ...

* Thiết kế Dashboard bằng Amazon QuickSight:
  * Kết nối Dataset
  * Tạo biểu đồ trực quan
  * Xây dựng báo cáo phân tích
  * ...

* Hiểu được quy trình xây dựng hệ thống Business Intelligence (BI) trên AWS từ khâu thu thập dữ liệu đến trực quan hóa kết quả.

* Có khả năng xây dựng một Data Analytics Pipeline hoàn chỉnh phục vụ cho việc phân tích và hỗ trợ ra quyết định.
