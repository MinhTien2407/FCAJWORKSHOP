### Kết quả đạt được tuần 1:

* Hiểu được quy trình xây dựng và triển khai hạ tầng cơ bản trên nền tảng AWS.

* Quản lý chi phí sử dụng AWS thông qua AWS Budgets:
  * Tạo Budget theo chi phí
  * Thiết lập ngưỡng cảnh báo
  * Theo dõi mức sử dụng dịch vụ
  * ...

* Nắm được cách quản lý người dùng và phân quyền với AWS IAM:
  * Tạo User
  * Tạo Group
  * Tạo Policy
  * Gán quyền theo nguyên tắc Least Privilege
  * ...

* Triển khai hạ tầng mạng với Amazon VPC:
  * Tạo VPC
  * Tạo Public và Private Subnet
  * Cấu hình Route Table
  * Internet Gateway
  * Security Group
  * ...

* Thực hành triển khai Amazon EC2:
  * Khởi tạo EC2 Instance
  * Kết nối SSH
  * Quản lý Security Group
  * Gắn Elastic IP
  * ...

* Hiểu và sử dụng IAM Role để cấp quyền cho ứng dụng truy cập các dịch vụ AWS mà không cần lưu trữ Access Key.

* Làm quen với môi trường phát triển trực tuyến AWS Cloud9.

* Triển khai website tĩnh với Amazon S3:
  * Upload website
  * Cấu hình Static Website Hosting
  * Phân quyền Bucket
  * ...

* Tạo và quản lý cơ sở dữ liệu với Amazon RDS:
  * Khởi tạo Database
  * Kết nối Database
  * Quản lý Endpoint
  * ...

* Tìm hiểu và sử dụng Amazon Lightsail để tối ưu chi phí triển khai máy chủ.

* Thực hành tự động mở rộng hệ thống với Amazon EC2 Auto Scaling.

* Thiết lập giám sát hệ thống bằng Amazon CloudWatch:
  * Theo dõi CPU
  * Theo dõi Memory
  * Thiết lập Alarm
  * ...

* Hiểu cách cấu hình hệ thống DNS với Amazon Route 53 và tích hợp giữa môi trường Local và Amazon VPC.

* Sử dụng AWS CLI trên Amazon EC2 để quản lý tài nguyên AWS.

* Hoàn thành workshop triển khai Highly Available Web Application:
  * Xây dựng hệ thống Web có tính sẵn sàng cao
  * Triển khai nhiều thành phần AWS phối hợp
  * Tăng khả năng chịu lỗi và mở rộng hệ thống
  * ...

* ...
