### Week 1 Objectives:

Become familiar with AWS core services and the process of deploying cloud infrastructure.

Understand how to build, manage, and operate AWS resources through hands-on practice.

Gain practical experience with AWS services for deploying a complete cloud application.

  * Configure budget alerts
  * Monitor AWS service usage
  * ...

  * Create IAM Users
  * Create IAM Groups
  * Create IAM Policies
  * Apply the Principle of Least Privilege
  * ...

  * Configure Public and Private Subnets
  * Configure Route Tables

  * ...

  * Connect via SSH
