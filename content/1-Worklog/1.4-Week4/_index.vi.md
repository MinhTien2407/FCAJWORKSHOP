### Mục tiêu tuần 4:

* Nâng cao kỹ năng quản lý và tự động hóa hạ tầng AWS thông qua AWS Systems Manager và AWS CloudFormation.

* Tìm hiểu các giải pháp tăng cường bảo mật và kiểm soát truy cập bằng IAM Permission Boundary, AWS WAF, AWS KMS và AWS Security Hub.

* Thực hành triển khai các giải pháp kết nối mạng và sao lưu dữ liệu nhằm đảm bảo tính sẵn sàng và khả năng phục hồi của hệ thống.

### Kết quả đạt được tuần 4:

* Hiểu được quy trình quản trị và tự động hóa hạ tầng trên AWS thông qua Infrastructure as Code.

* Thực hành triển khai AWS Systems Manager:
  * Quản lý máy chủ tập trung
  * Truy cập EC2 bằng Session Manager
  * Thực hiện các tác vụ quản trị từ xa
  * ...

* Nắm được cách triển khai Infrastructure as Code bằng AWS CloudFormation:
  * Tạo CloudFormation Template
  * Triển khai Stack
  * Cập nhật hạ tầng
  * ...

* Tăng cường khả năng quản lý tài nguyên trên AWS:
  * Quản lý tài nguyên bằng Tag
  * Tổ chức Resource Groups
  * Kiểm soát quyền truy cập theo Tag
  * ...

* Hiểu và áp dụng các cơ chế bảo mật trên AWS:
  * Thiết lập IAM Permission Boundary
  * Giới hạn Assume Role
  * Kiểm soát quyền truy cập người dùng
  * ...

* Thực hành triển khai các dịch vụ bảo mật:
  * Đánh giá bảo mật bằng AWS Security Hub
  * Bảo vệ ứng dụng với AWS WAF
  * Quản lý khóa mã hóa bằng AWS KMS
  * ...

* Thiết lập giải pháp sao lưu dữ liệu bằng AWS Backup:
  * Xây dựng Backup Plan
  * Cấu hình Backup Vault
  * Khôi phục dữ liệu từ Backup
  * ...

* Hiểu cách xây dựng kiến trúc mạng linh hoạt trên AWS:
  * Kết nối VPC bằng VPC Peering
  * Quản lý định tuyến với AWS Transit Gateway
  * Tăng khả năng mở rộng hệ thống
  * ...

* Nâng cao kỹ năng quản trị, bảo mật và vận hành hệ thống theo các tiêu chuẩn triển khai trên AWS.

* Hoàn thành workshop về quản trị hệ thống và bảo mật trên AWS:
  * Tự động hóa quản lý hạ tầng
  * Tăng cường bảo mật và phân quyền
  * Triển khai giải pháp sao lưu dữ liệu
  * Xây dựng kiến trúc mạng linh hoạt
  * ...
