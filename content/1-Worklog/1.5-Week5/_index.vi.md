* Hiểu được quy trình triển khai ứng dụng container hóa trên nền tảng AWS.

* Thực hành triển khai Docker và Amazon ECS:
  * Đóng gói ứng dụng bằng Docker
  * Triển khai Container trên Amazon ECS
  * Quản lý dịch vụ Container
  * ...

* Xây dựng hệ thống CI/CD bằng AWS CodePipeline:
  * Tự động Build ứng dụng
  * Tự động triển khai
  * Giảm thiểu thao tác thủ công
  * ...

* Tìm hiểu các giải pháp lưu trữ và quản lý dữ liệu:

  * Xây dựng Data Lake trên AWS
  * ...

  * Thiết kế Partition Key
  * Quản lý dữ liệu hiệu quả
  * ...

  * Savings Plans

  * Reserved DB Instances
  * ...

  * Phân tích Cost and Usage Reports (CUR)
  * ...
