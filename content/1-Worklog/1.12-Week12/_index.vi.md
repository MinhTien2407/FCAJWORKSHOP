## Tuần 12: Phát triển Ứng dụng Serverless trên AWS (Từ ngày 13/07/2026 đến ngày 15/07/2026)

### A. Mục tiêu

* Hiểu kiến trúc và quy trình phát triển ứng dụng Serverless trên nền tảng AWS.
* Xây dựng ứng dụng Serverless hoàn chỉnh sử dụng AWS Lambda, Amazon API Gateway, Amazon S3 và Amazon DynamoDB.
* Triển khai ứng dụng Serverless bằng AWS Serverless Application Model (AWS SAM).
* Thiết lập xác thực người dùng với Amazon Cognito và bảo mật ứng dụng bằng chứng chỉ SSL.

### C. Kết quả đạt được

* Hiểu được kiến trúc và quy trình xây dựng ứng dụng Serverless trên nền tảng AWS.

* Xây dựng ứng dụng Serverless với AWS Lambda:
  * Xử lý nghiệp vụ bằng Lambda Function
  * Kết nối và lưu trữ dữ liệu trên Amazon S3
  * Thực hiện các thao tác với Amazon DynamoDB
  * ...

* Xây dựng Frontend tích hợp với Amazon API Gateway:
  * Gọi REST API
  * Trao đổi dữ liệu giữa Frontend và Backend
  * Kiểm thử luồng xử lý dữ liệu
  * ...

* Triển khai ứng dụng bằng AWS Serverless Application Model (AWS SAM):
  * Đóng gói ứng dụng
  * Triển khai hạ tầng bằng Infrastructure as Code
  * Quản lý và cập nhật ứng dụng
  * ...

* Thiết lập hệ thống xác thực người dùng với Amazon Cognito:
  * Tạo User Pool
  * Quản lý người dùng
  * Bảo vệ quyền truy cập ứng dụng
  * ...

* Cấu hình chứng chỉ SSL cho ứng dụng:
  * Thiết lập kết nối HTTPS
  * Bảo mật dữ liệu truyền tải
  * Nâng cao mức độ an toàn cho hệ thống
  * ...

* Có khả năng tích hợp nhiều dịch vụ Serverless của AWS để xây dựng một ứng dụng hoàn chỉnh.

* Nắm được quy trình phát triển, triển khai và bảo mật ứng dụng Serverless theo các Best Practices trên nền tảng AWS.
