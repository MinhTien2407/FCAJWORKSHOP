### Mục tiêu tuần 6:

* Hiểu và triển khai kiến trúc Serverless trên AWS với sự kết hợp của Lambda, API Gateway, Amazon S3 và Amazon DynamoDB.

* Nắm được các phương pháp triển khai, bảo mật và quản lý ứng dụng Serverless bằng AWS SAM, Amazon Cognito và chứng chỉ SSL.

* Thực hành xây dựng hệ thống xử lý bất đồng bộ, tự động hóa quy trình CI/CD và giám sát hiệu suất ứng dụng trên AWS.

### Kết quả đạt được tuần 6:

* Hiểu được kiến trúc Serverless và quy trình triển khai ứng dụng trên AWS.

* Thực hành xây dựng ứng dụng Serverless:
  * Kết nối Amazon API Gateway với AWS Lambda
  * Tương tác dữ liệu với Amazon S3
  * Làm việc với Amazon DynamoDB
  * ...

* Triển khai ứng dụng bằng AWS SAM:
  * Đóng gói ứng dụng Serverless
  * Triển khai hạ tầng bằng Infrastructure as Code
  * Quản lý phiên bản ứng dụng
  * ...

* Thiết lập cơ chế xác thực và bảo mật:
  * Quản lý người dùng với Amazon Cognito
  * Cấu hình chứng chỉ SSL
  * Tăng cường bảo mật ứng dụng
  * ...

* Xây dựng hệ thống xử lý bất đồng bộ:
  * Gửi thông điệp bằng Amazon SNS
  * Xử lý hàng đợi với Amazon SQS
  * Tăng khả năng mở rộng hệ thống
  * ...

* Làm quen với AWS AppSync:
  * Tìm hiểu GraphQL API
  * Đồng bộ dữ liệu theo thời gian thực
  * ...

* Tự động hóa quy trình triển khai bằng AWS CodePipeline.

* Thiết lập giám sát và phân tích hệ thống:
  * Theo dõi log bằng Amazon CloudWatch
  * Phân tích luồng xử lý với AWS X-Ray
  * Hỗ trợ phát hiện và xử lý lỗi
  * ...

* Nâng cao kỹ năng phát triển, triển khai và vận hành ứng dụng Serverless trên nền tảng AWS.

* Hoàn thành workshop về Serverless Application:
  * Xây dựng Backend Serverless
  * Triển khai ứng dụng bằng AWS SAM
  * Thiết lập CI/CD
  * Giám sát và tối ưu hiệu suất hệ thống
  * ...
