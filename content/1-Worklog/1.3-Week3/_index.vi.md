### Kết quả đạt được tuần 3:

* Hiểu được các nguyên tắc tối ưu hóa hệ thống theo AWS Well-Architected Framework.

* Nắm được các phương pháp nâng cao hiệu quả vận hành hệ thống trên AWS:
  * Tự động hóa quy trình vận hành
  * Giám sát hệ thống
  * Quản lý tài nguyên
  * ...

* Thực hành xây dựng quy trình tự động hóa bằng AWS Lambda:
  * Tự động tắt EC2 theo lịch
  * Gửi thông báo đến Slack
  * Kiểm tra kết quả thực thi
  * ...

* Thiết lập hệ thống giám sát bằng Amazon CloudWatch và Grafana:
  * Thu thập các chỉ số hoạt động của hệ thống
  * Xây dựng Dashboard trực quan
  * Theo dõi trạng thái tài nguyên AWS
  * ...

* Tìm hiểu cách quản lý tài nguyên bằng Tag và Resource Groups:
  * Gắn Tag cho tài nguyên
  * Tổ chức Resource Groups
  * Quản lý tài nguyên theo từng nhóm
  * ...

* Thực hành quản lý quyền truy cập Amazon EC2 thông qua IAM kết hợp Tag.

* Làm quen với AWS Systems Manager:
  * Quản lý máy chủ tập trung
  * Thực hiện Automation
  * Quản lý cấu hình hệ thống
  * ...

* Thực hành sử dụng AWS Systems Manager Session Manager:
  * Truy cập EC2 không cần SSH
  * Quản lý phiên làm việc an toàn
  * Ghi nhận lịch sử phiên kết nối
  * ...

* Hiểu được lợi ích của Infrastructure as Code (IaC) trong việc triển khai và quản lý hạ tầng.

* Thực hành triển khai hạ tầng bằng AWS CloudFormation:
  * Tạo CloudFormation Template
  * Triển khai Stack
  * Cập nhật Stack
  * Xóa Stack
  * ...

* Nắm được quy trình tự động hóa triển khai và quản lý hạ tầng trên AWS.

* Hoàn thành workshop tối ưu hóa hệ thống trên AWS:
  * Tự động hóa vận hành hệ thống bằng AWS Lambda
  * Thiết lập giám sát và trực quan hóa dữ liệu với CloudWatch và Grafana
  * Quản lý tài nguyên bằng Tag và Resource Groups
  * Quản trị máy chủ với AWS Systems Manager
  * Triển khai hạ tầng bằng AWS CloudFormation
  * ...

* ...
